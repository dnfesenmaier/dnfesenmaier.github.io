// How I Work
01
Discover

Start with the business question, not the dataset. Talk to stakeholders, find the decision being made, and reverse-engineer the data that proves or disproves the hypothesis.

02
Model

Build the SQL and Python that holds up to production load. Customer360 datasets, multi-touch attribution, propensity models — engineered to be queried, joined, and trusted.

03
Ship

Get insights in front of the people making decisions. Tableau dashboards, A/B test readouts, exec memos — designed for the audience, not the analyst.

04
Iterate

Measure what shipped, learn from feedback loops, refine models. LLM classifiers improve with RAG; propensity models retrain on new conversions; dashboards evolve with the org.
